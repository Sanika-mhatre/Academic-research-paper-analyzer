 
# 📄 Academic Research Paper Analyzer

A Streamlit-based web application that analyzes research papers using Machine Learning and NLP techniques.

## 🚀 Features
- 📊 Novelty & Clarity prediction (ML models)
- 📝 Grammar checking (rule-based NLP)
- 🔍 Plagiarism detection (TF-IDF + cosine similarity)
- 🌐 Copyleaks API integration
- 📈 Dashboard with readiness score
- 💬 User feedback system

## 🧠 Tech Stack
- Python
- Streamlit
- Scikit-learn
- MongoDB
- Copyleaks API

## ▶️ Run Locally
```bash
pip install -r requirements.txt
python -m streamlit run app.py
